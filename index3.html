クラス型オブジェクト

classによるクラス定義

var 変数 : クラス = new クラス ( 引数 );
クラスというオブジェクトを作成し、これをもとにオブジェクトを作成する
クラスはインスタンスを利用する。インスタンスはクラスを実体化したオブジェクト
クラスの中に保管できるもの
コンストラクタ：インスタンスを作成する際の初期化処理を用意する特別なメソッド
プロパティ：オブジェクト内に値を保持するための変数
メソッド：オブジェクト内に用意される関数

アクセス修飾子

  プロパティやメソッドがどの範囲からアクセス可能か指定する
  public：どこからでもアクセスできる
  private：クラスの中からしかアクセスできません
  protected：基本的にprivateと同じだが、「継承」利用の場合、サブクラスからもアクセスできる
  class Person {
    private name:string;
    protected age:number;

    constructor(n:string, y:number) {
      ......
    }
    public print():string {
      ......
    }
  }

クラスの継承

継承は既にあるクラスのすべての機能を引き継いで新しいクラスを定義する
既に作成したクラスよ有効活用できる
class クラス extends 継承するクラス {....}
